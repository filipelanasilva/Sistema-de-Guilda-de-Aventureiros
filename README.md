# Sistema de Guilda de Aventureiros

Sistema de gerenciamento de uma Guilda de Aventureiros desenvolvido em Java com o objetivo de praticar os fundamentos da linguagem e os conceitos de Programação Orientada a Objetos.

O projeto está sendo desenvolvido de forma incremental, recebendo novas funcionalidades conforme novos conteúdos são estudados.

---

## Funcionalidades

### Gerenciamento de aventureiros

- Cadastro de aventureiros
- Busca de aventureiros
- Edição de aventureiros
- Remoção de aventureiros
- Listagem de aventureiros

### Formas de listagem

- Por classe
- Por nível
- Por idade
- Por rank

### Validações

- Classe válida
- Rank válido
- Faixa de nível permitida
- Idade mínima
- Controle de limite do vetor de aventureiros
- Validação de opções do menu

## Estrutura do projeto

```
src
└── com
    └── filipelana
        └── guilda
            ├── application
            ├── model
            ├── service
            └── util
```

### application

Responsável pelo ponto de entrada da aplicação.

### model

Contém as entidades do sistema.

- Aventureiro
- Guilda

### service

Contém toda a lógica de negócio da aplicação.

### util

Classes auxiliares para entrada de dados e navegação dos menus.

## Objetivo do projeto

Este projeto tem como principal objetivo consolidar os conhecimentos adquiridos durante os estudos de Java através do desenvolvimento de uma aplicação prática.

Ao invés de criar diversos projetos pequenos, a proposta é evoluir continuamente este sistema, incorporando novos conceitos à medida que são aprendidos.
